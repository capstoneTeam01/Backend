## Presentation Standard

Each image uses a square canvas, white studio background, centred product framing, soft lighting, and generic unbranded equipment without logos, labels, packaging, or watermarks. FixBee converts the generated PNG previews to high-quality JPEG assets while preserving the existing application filenames and URL mappings.
